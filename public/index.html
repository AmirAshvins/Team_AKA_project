Features

Our website is a group-moderated assignment tracker for Computer Systems Technology (CST) students from BCIT.


Each user has an interactive list of all the assignments that have been added. From that list, users can click on assignments to view or edit the assignment's details.


Each user has the ability to add and edit assignments to the database. These assignments will then be visible to every other user.


Each user has the ability to cross off any of their completed assignments from the list. The crossed off assignments are unique to each user and will only appear to be crossed off on the user's own list.
